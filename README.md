#  2D Dino Character Movement (Unity)

A simple 2D character controller for a Dino character in Unity with
idle, walk, and jump animations.

##  Features
- Left / Right movement
- Jump mechanic
- Animator-based state transitions
- Idle, Walk, Jump animations


##  Tools Used
- Unity (2D)
- C#
- Animator Controller
- Sprite animations
